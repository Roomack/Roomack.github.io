Git Commands
Create a new local repository: git init
Clone an existing repository: git clone ssh://user@domain.com/repo.git
Check file status of directory: git status
Add all changed files: git add .
Add a specific file: git add -p file
Commit changes: git commit -m message
Amend the last commit: git commit --amend
List all existing branches: git branch
Create a new branch: git branch new-branch
Delete a local branch: git branch -d branch
List all currently configured remotes: git remote -v
Download all changes from remote: git fetch remote
Download changes and integrate into head: git pull remote branch
Publish local changes to remote: git push remote branch
Merge branch into head: git merge branch
Discard all local changes in dir: git reset --hard HEAD
Merge branch into head: git merge branch
